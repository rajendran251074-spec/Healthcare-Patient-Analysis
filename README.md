 ## Project Overview
Analyzed real healthcare patient data using MySQL and , Power BI
to uncover insights on patient demographics, encounters,
procedures, and insurance coverage.

##  Tools Used
  **MySQL Workbench** — Data cleaning, querying, and analysis  
  **Power BI Desktop** — Interactive dashboard creation and visualization  
  **Microsoft Excel** — Data source and preprocessing
  
##  Dataset
 974 Patients
 27,891 Encounters
 47,701 Procedures
 10 Insurance Payers
 1 Organization

##  Key Insights
 Boston has highest patients (541 - 55%)
 Medicare provides highest coverage ($19.2M)
 Inpatient most expensive ($7,761 avg cost)
 NO_INSURANCE — 8,807 encounters, $0 coverage
 Renal Dialysis — most common procedure (2,746)
